## 1. 왜 Beyond AI인가

### AI 시대, 기업교육의 새로운 질문

```
"AI가 더 잘하는 시대에, 우리 조직원의 경쟁력은 무엇인가?"
```

AI는 빠르고 정확합니다. 하지만 위기 상황의 즉각적 판단, 팀원 간 갈등 조정, 고객의 숨은 니즈를 읽는 직관은 여전히 사람의 영역입니다.

Beyond AI는 이 질문에 대한 답을 **12개월간의 실전 대결**로 찾아갑니다.

### 기존 기업교육의 한계

| 기존 교육 | Beyond AI |
|-----------|-----------|
| 일방향 강의 | 쌍방향 대결 |
| 이론 중심 | 실전 미션 중심 |
| 정성적 만족도 평가 | 4축 정량 평가 |
| 단발성 교육 | 12개월 연속 여정 |
| 낮은 참여율 (35%) | 높은 참여율 (85%+) |
| 성과 측정 불가 | 데이터 기반 성장 측정 |

## 2. 프로그램 개요

### 핵심 콘셉트

**사람과 AI가 동일한 직무 과제를 동시에 수행하고, 실시간으로 점수를 비교합니다.**

- 기간: 12개월 (월 1회 메인 퀘스트 + 일일 미니 챌린지)
- 대상: 기업 임직원 (직급/직무 무관)

- 결과: 개인별/팀별 역량 성장 리포트

## 3. 배틀 시스템

### 대결 프로세스

```
1. 퀘스트 오픈 → 미션 확인
2. 동시 수행 → 사람과 AI가 같은 과제 수행
3. AI 사고 과정 → 실시간 공개 (학습 효과)
4. 자동 평가 → 4축 정량 채점
5. 결과 비교 → 항목별 승패 및 점수
```

### 평가 기준

| 항목 | 비중 | 설명 |
|------|------|------|
| 품질 (Quality) | 30% | 결과물의 완성도와 정확성 |
| 창의성 (Creativity) | 25% | 독창적 접근과 차별화된 관점 |
| 실행력 (Execution) | 25% | 구체적이고 실행 가능한 내용 |
| 효율성 (Efficiency) | 20% | 제한 시간 내 효과적 수행 |

### 적응형 난이도

학습자의 실력에 따라 AI의 난이도가 자동 조절됩니다.
- 승률 70% 이상 → AI 난이도 상향
- 승률 30% 이하 → AI 난이도 하향
- 모든 수준의 학습자가 적절한 도전감을 유지

- **Gold**: 전문가의 영역
- **Platinum**: 전설의 시작

### 타이틀 (7단계)
ROOKIE → CHALLENGER → WARRIOR → STRATEGIST → LEGEND → AI SLAYER → UNSTOPPABLE

### 포인트 & 리더보드

- 대결 참여: 기본 포인트 지급 (패배 시에도)

- 7일 이상 연속 → 보너스 포인트 배수

## 5. 도입 효과

### 정량적 효과

| 지표 | 기존 교육 | Beyond AI |
|------|-----------|-----------|
| 참여율 | 35% | 85%+ |
| 지식 유지율 (20일 후) | 10% | 90% |
| 교육 완료율 | 50% | 92% |
| 역량 성장률 (12개월) | 측정 불가 | 평균 35% |

### 정성적 효과

- **AI 리터러시 향상**: AI의 강점과 한계를 체감 수준으로 이해
- **자기 인식 강화**: 자신만의 경쟁력과 개선 영역 명확화
- **학습 문화 정착**: 게이미피케이션으로 자발적 학습 습관 형성
- **팀 결속력**: 팀 리더보드를 통한 건전한 경쟁과 협력

## 6. 관리자 대시보드

기업 교육 담당자를 위한 전용 관리 화면을 제공합니다.

### 핵심 기능
- **참여 현황**: 실시간 참여율, 완료율, 활성 사용자 수
