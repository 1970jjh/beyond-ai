## 1월 미니 챌린지: 데이터 해석 훈련 시리즈

### MC-01-A: 뉴스 인사이트 추출 (매주 월요일, 5분)

**방식**: 플랫폼이 당일 경제/산업 뉴스 헤드라인 1건 제공
**과제**: 이 뉴스에서 **비즈니스 인사이트 3가지**를 도출하세요
**AI도 동시에 수행 → 즉시 비교**

```
예시 뉴스: "국내 펫푸드 시장 5년 내 2배 성장 전망, 프리미엄 브랜드 점유율 급증"

참가자 예시 답:
1. [시장 기회] 프리미엄 시장 진입은 지금이 적기 — 선도 기업이 아직 고착화 전
2. [위협 요인] 대기업 진입 가능성 높아 — 1-2년 내 선점 필수
3. [인간 인사이트] 반려동물 없는 사람도 선물용으로 구매 — 타겟 더 넓음

AI 예시 답:
1. 시장 CAGR 약 14% 추정, 2030년 시장 규모 약 1.5조원
2. 프리미엄 세그먼트가 전체 성장의 70% 이상 기여 예상
3. 해외 직구 증가로 국내 로컬 브랜드 차별화 필요성 증가
```

**채점 포인트**: AI는 수치 중심, 사람은 맥락 & 행동 인사이트 중심
**이긴다는 기준**: "이 인사이트로 지금 당장 의사결정할 수 있는가"

---

### MC-01-B: 차트 읽기 챌린지 (매주 수요일, 5분)

**방식**: 복잡한 차트 이미지 1개 제공 (꺾은선/막대/산점도 등)
**과제**:
1. 이 차트가 말하는 것 (사실)
2. 이 차트가 숨기는 것 (한계)
3. 이 차트 기반 의사결정 1개 (행동)

**보너스 포인트**: "이 차트에서 아무도 주목 안 한 것" 발견 시

---

### MC-01-C: 숫자 감각 훈련 (격주 금요일, 3분)

**방식**: 숫자 하나 제시 (예: "한국인의 하루 평균 유튜브 시청 시간은?")
**과제**: AI 없이 직관으로 추정 → AI 정답과 비교
**목표**: 틀려도 됨. 왜 그렇게 생각했는지가 중요

## 2월 미니 챌린지: 관찰 & 공감 훈련 시리즈

### MC-02-A: 1분 페르소나 (매주 월요일, 10분)

**방식**: 주변 실제 인물 1명을 관찰하고 페르소나 스케치
**과제**:
- 이름(가명):
- 지금 이 사람의 가장 큰 불편함:
- 이 사람이 절대 말 못할 속마음:
- 이 사람이 즐겨 쓰는 앱 3개 추측:

**AI도 같은 직업/상황 설정으로 페르소나 생성**
**비교 포인트**: AI는 통계적으로 맞지만 당신의 페르소나가 더 "살아있는가"

---

### MC-02-B: 공감 언어 변환 (매주 수요일, 5분)

**방식**: 사무적인 문장 1개 제공
**과제**: 같은 내용을 **공감이 느껴지는 언어로** 바꾸세요

```
예시:
원문: "귀하의 주문 번호 #38291이 배송 지연되었습니다. 3-5일 내 발송 예정입니다."

사람 변환 예:
"중요한 물건 기다리고 계셨을 텐데 죄송합니다. [이름]님 주문이 예상보다 늦어지고 있어요.
현재 확인해 보니 [날짜]까지는 반드시 도착할 수 있을 것 같습니다."

AI 변환 예:
"주문 #38291 배송이 지연되어 불편을 드린 점 사과드립니다. 3-5영업일 내
발송 예정이며, 발송 시 이메일로 안내드리겠습니다."
```

**비교 포인트**: 어느 쪽이 더 "사람답게" 느껴지는가

---

### MC-02-C: 욕구 계층 분석 (격주 금요일, 10분)

**방식**: 고객 불만 사례 1개 제공
**과제**: 이 불만의 진짜 욕구를 4단계로 분석

```
레벨 1 (표면): 고객이 말한 것
레벨 2 (행동): 고객이 실제로 한 것
레벨 3 (감정): 고객이 느끼는 것
레벨 4 (욕구): 고객이 진짜 원하는 것
```

## 3월 미니 챌린지: 논리 구조 훈련 시리즈

### MC-03-A: 엘리베이터 피치 (매주 월요일, 5분)

**방식**: 아이디어/아이템 하나 제시 (또는 자유 선택)
**과제**: 30초 엘리베이터 피치 작성

```
피치 구조 (30초 = 75단어):
[후킹 질문 1문장]
[문제 1문장]
[솔루션 1문장]
[차별점 1문장]
[행동 촉구 1문장]
```

**AI도 동일 주제로 30초 피치 생성**
**승부처**: 어느 쪽이 더 "듣고 싶다"는 느낌인가

---

### MC-03-B: 주장 강화하기 (매주 수요일, 5분)

**방식**: 약한 주장 1개 제시
**과제**: 같은 주장을 3배 더 설득력 있게 만드세요

```
원문 (약한 주장):
"우리 제품이 더 좋습니다. 기능이 많고 가격도 합리적입니다."

강화 방향 힌트:
- 수치를 넣으세요
- 고객의 언어를 쓰세요
- 감정을 건드리세요
- 리스크를 제거하세요
```

---

### MC-03-C: 반론 대응 훈련 (격주 금요일, 10분)

**방식**: 제안서 1개 + 반론 3가지 제시
**과제**: 각 반론에 대한 설득력 있는 대응 작성

```
반론 유형:
1. "너무 비싸요" → _______
2. "지금은 타이밍이 아니에요" → _______
3. "이미 다른 솔루션 쓰고 있어요" → _______
```

**AI도 동일 반론 대응**
**비교 포인트**: AI는 논리적, 사람은 감정적 설득도 가능
