## 4월 미니 챌린지: 의사결정 속도 훈련 시리즈

### MC-04-A: 30초 딜레마 (매주 월요일, 5분)

**방식**: 업무 딜레마 상황 카드 1개 제시
**과제**: 30초 안에 결정하고 이유를 한 문장으로

```
딜레마 카드 예시:
"팀원이 실수를 했습니다. 경위서를 쓰게 하면 책임감을 갖겠지만
팀 분위기가 나빠질 수 있습니다. 어떻게 하시겠습니까?"

선택: [경위서 O / X / 다른 방법]
이유: _______ (1문장)
```

**AI도 동일 딜레마에 즉시 응답**
**승부처**: AI는 "양측 모두 장단점..." 으로 회피. 당신은 결단하세요.

**딜레마 카드 풀 (월별 로테이션)**:
1. 중요한 미팅 vs 아픈 부모님 — 어떻게?
2. 팀장 의견이 틀렸는데 발표 자료는 완성됐다 — 어떻게?
3. 동료가 비윤리적 업무를 지시받았다고 털어놓는다 — 어떻게?
4. 핵심 고객이 무리한 요구를 한다 — 들어주면 관계는 OK, 안 들어주면 신뢰 — 어떻게?

### MC-04-B: 위기 메시지 30초 드래프트 (매주 수요일, 8분)

**방식**: 미니 위기 상황 제시 (SNS 1건, 짧은 상황 설명)
**과제**: 30초 안에 첫 번째 공식 응답 초안 (2-3문장)

```
상황 예시:
[SNS 트윗] "방금 @[브랜드] 앱이 갑자기 터지면서 작성하던 보고서가 다 날아갔어요.
내일 임원 보고인데 진짜 막막합니다... 이거 보상은 어떻게 되는 건가요?"

당신의 첫 응답:
"______________________________________________"
```

**평가**: 공감 + 책임인정 + 다음 단계 안내가 모두 있는가?

**방식**: 이번 주 가장 어려웠던 의사결정 1개 회고
**과제**:

**목적**: 의사결정 패턴 자기인식 + 12월 연간 성장 리포트 소재

## 5월 미니 챌린지: 공감 & 경청 훈련 시리즈

### MC-05-A: 팀원 스토리 캐치 (매주 월요일, 10분)

**방식**: 가상 팀원의 짧은 이야기 제공 (2-3문장)
**과제**:
1. 이 사람이 말하지 않은 감정: _______
2. 이 사람이 진짜 원하는 것: _______
3. 당신이라면 다음에 어떤 말을 할 것인가: _______

```
팀원 이야기 예시:
"요즘 좀 지치네요. 열심히 하는데 성과가 안 보이는 것 같아서...
뭐 그냥 제 문제겠죠."

AI 응답 예: "번아웃 징후가 관찰됩니다. 휴식과 업무 재설계를 권장합니다."
사람 응답 방향: ??
```

**포인트**: AI의 응답이 "맞는 말"이어도 왜 차갑게 느껴지는가?

---

### MC-05-B: 팀 활동 아이디어 5분 브레인스토밍 (매주 수요일, 5분)

**방식**: 팀 현황 카드 제시 (팀 규모, 분위기, 문제)
**과제**: 5분 안에 아이디어 5개 (판단 금지)
**규칙**: 비현실적이어도 OK, 이상해도 OK

**AI 아이디어와 비교 시 체크 포인트**:
- AI가 나열한 활동 중 "우리 팀에 맞는" 것이 있는가?
- 당신의 아이디어 중 AI가 절대 못 낼 것이 있는가?

### MC-05-C: 갈등 뒤 관계 복원 메시지 (격주 금요일, 10분)

**방식**: 갈등 후 상황 카드 제시
**과제**: 관계 복원을 위한 첫 메시지 작성 (카카오톡 형식)

```
상황: 어제 회의에서 당신이 김대리의 의견을 공개적으로 반박했습니다.
김대리가 조용해졌고, 오늘 출근 후 눈을 피하는 것 같습니다.

당신의 메시지:
"[어제 회의] _________________________"
```

**AI 메시지와 비교**: 어느 쪽이 더 "진심"처럼 읽히는가?

## 6월 미니 챌린지: 설득 언어 훈련 시리즈

### MC-06-A: 같은 내용 3가지 버전 (매주 월요일, 10분)

**방식**: 메시지 1개 제시
**과제**: 동일 내용을 **3가지 다른 청중**에 맞게 변환

```
원본 메시지:
"우리 시스템 업그레이드로 처리 속도가 40% 향상되었습니다."

버전 A (CEO에게): _______
버전 B (현장 직원에게): _______
버전 C (고객에게): _______
```

**포인트**: AI는 3버전을 정확히 만들지만, 당신의 버전이 더 "그 사람 언어"인가?

---

### MC-06-B: 오프닝 훅 10가지 (매주 수요일, 5분)

**방식**: 발표 주제 1개 제시
**과제**: 오프닝 훅 10가지 빠르게 나열 (판단 금지)

```
주제: "우리 팀의 올해 목표 발표"

훅 예시들:
1. 충격적 통계: "여러분, 작년에..."
2. 질문: "만약 지금 당장..."
3. 스토리: "3개월 전, 팀원 한 명이..."
4. 도발: "많은 팀들이 목표를 세웁니다. 그리고 대부분 실패합니다."
5-10: (직접 작성)
```

---

### MC-06-C: 침묵의 힘 (격주 금요일, 5분)

**방식**: 발표 스크립트 1개 제시
**과제**: "침묵을 넣어야 할 3곳"을 표시하고 이유 설명

```
예시 스크립트:
"우리는 올해 목표를 달성했습니다. [?] 하지만 그 과정은 쉽지 않았습니다. [?]
팀원 모두가 포기하고 싶었던 순간이 있었습니다. [?] 오늘 그 이야기를 하려 합니다."
```

**포인트**: AI는 침묵의 타이밍을 모릅니다. 발표는 "말"만이 아닙니다.
