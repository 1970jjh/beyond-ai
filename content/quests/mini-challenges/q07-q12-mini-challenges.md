## 7월 미니 챌린지: 관찰 훈련 시리즈

### MC-07-A: 오늘의 낭비 찾기 (매주 월요일, 5분)

**방식**: 오늘 업무 중 "시간 낭비"를 1가지 발견
**과제**:
```
낭비 항목: _______
예상 낭비 시간: 주 ___ 시간 × 52주 = 연 ___ 시간
개선 아이디어: _______
실행 장벽: _______
```
**누적**: 7월 내 발견한 낭비 목록 → 메인 퀘스트 실전 소재로 활용

---

### MC-07-B: 프로세스 한 줄 지도 (매주 수요일, 10분)

**방식**: 자신의 반복 업무 1가지 선택
**과제**: 그 프로세스를 "한 줄 흐름도"로 표현
```
예: 이메일 보고 → 부장 검토 → 수정 → 재검토 → 발송
       (5분)      (2일)     (1시간)  (1일)    (5분)
```
**AI와 비교**: AI의 추천 개선 프로세스 vs 당신의 아이디어

---

### MC-07-C: "왜?" 5번 훈련 (격주 금요일, 5분)

**방식**: 업무 문제 상황 1개
**과제**: "왜?"를 5번 반복하여 근본 원인 도출
```
문제: 고객 불만이 증가하고 있다
왜1? 배송이 늦어지기 때문
왜2? 물류 파트너 처리 속도가 느려서
왜3? 우리가 갑자기 주문량을 늘려서
왜4? 마케팅 캠페인 성과가 예상보다 좋아서
왜5? 시스템이 급증한 주문량을 감당 못해서
근본 원인: 시스템 용량 계획 부재
```

## 8월 미니 챌린지: 감정 인식 훈련 시리즈

### MC-08-A: 감정 온도 읽기 (매주 월요일, 5분)

**방식**: 고객/동료의 짧은 메시지 제시
**과제**: 이 메시지의 "감정 온도"를 분석
```
감정 온도 척도:
🔴 100 (분노/폭발)
🟠 80 (불만)
🟡 60 (불편)
🟢 40 (중립)
🔵 20 (긍정)

메시지: "... 네, 알겠습니다."
온도: ___°
근거: _______
적절한 대응 방식: _______
```
**포인트**: 같은 "알겠습니다"도 온도가 다릅니다. AI는 이 차이를 모릅니다.

---

### MC-08-B: 공감 → 해결 순서 연습 (매주 수요일, 8분)

**방식**: 불만 상황 카드
**과제**: 해결책을 제시하기 전에 반드시 공감 먼저

```
규칙: "해결책 제시" 전 최소 2번의 공감 표현 포함

나쁜 예:
"죄송합니다. 환불처리는 3-5일 소요됩니다."

좋은 예:
"많이 당황하셨겠어요. [공감1]
기다리고 계신 상황에서 이런 일이 생겨서 정말 죄송합니다. [공감2]
빠른 처리를 위해 제가 직접 확인해 드리겠습니다. [해결]"
```

## 9월 미니 챌린지: 수치 직관 훈련 시리즈

### MC-09-A: 숫자 뒤 이야기 (매주 월요일, 5분)

**방식**: 지표 하나 제시 (예: "이탈률 15% → 25% 상승")
**과제**: 이 숫자 변화 뒤에 있을 수 있는 이야기 3가지

```
숫자: 고객 이탈률 15% → 25% (3개월 사이)
이야기 #1: 경쟁사가 프로모션을 했을 것이다
이야기 #2: 우리 가격을 인상한 직후다
이야기 #3: 핵심 기능에 버그가 있었을 것이다
```
**AI 버전과 비교**: AI는 가설을 나열, 당신은 맥락이 있는 이야기

---

### MC-09-B: 대시보드 쓸모없는 지표 찾기 (매주 수요일, 5분)

**방식**: 대시보드 스냅샷 제시 (가상)
**과제**: 이 중 "의사결정에 실제로 쓸모없는 지표"를 골라내고 이유 설명

**포인트**: 데이터 리터러시의 핵심은 더 많은 데이터가 아니라 **올바른 데이터**입니다.

## 10월 미니 챌린지: 창의적 발산 훈련 시리즈

### MC-10-A: 랜덤 조합 아이디어 (매주 월요일, 5분)

**방식**: 완전히 관계 없는 단어 2개 제시
**과제**: 이 두 개를 결합한 비즈니스 아이디어 5개 (판단 금지!)

```
예: "세탁소" + "구독"
1. 매월 정기 세탁 픽업/배달 구독
2. 세탁 전문가와 패션 코디 구독 결합
3. 세탁소 유휴 공간 공유 오피스 구독
4. AI 얼룩 분석 + 세탁법 구독 앱
5. 세탁 진행 현황 실시간 알림 구독
```
**AI와 비교**: 아이디어 수는 비슷하지만 "이 중 실제로 해보고 싶은 것"은?

---

### MC-10-B: 기존 제품의 새 용도 (매주 수요일, 5분)

**방식**: 제품/서비스 1개 + 전혀 다른 타겟 제시
**과제**: 이 타겟에게 맞는 새로운 포지셔닝 3가지

```
예: 기업 교육 플랫폼 → 실버 세대용으로
1. "손주와 함께 배우는 디지털 능력" 포지셔닝
2. "퇴직 후 재취업 역량 강화" 포지셔닝
3. "시니어 기업인의 AI 리더십" 포지셔닝
```

## 11월 미니 챌린지: 관점 전환 훈련 시리즈

### MC-11-A: 상대 입장 대변 (매주 월요일, 10분)

**방식**: 갈등 상황 카드 + 내가 반대하는 입장 배정
**과제**: 내가 동의하지 않는 입장을 **최대한 설득력 있게** 대변하세요

```
규칙: 자신의 실제 의견 금지
"이 입장에서 가장 강력한 주장 3가지는 무엇인가?"

예: "야근은 필요하다" (당신이 동의하지 않는 입장)
주장1: _______
주장2: _______
주장3: _______
```
**목적**: 갈등 중재자가 되려면 내가 동의하지 않는 것도 이해해야 합니다.

---

### MC-11-B: Win-Win 프레임 연습 (매주 수요일, 10분)

**방식**: 협상/갈등 상황 카드
**과제**: 양측이 모두 "좋다"고 할 수 있는 창의적 해결책 1개

```
규칙: 타협(반반 나누기) 금지. 새로운 방법 찾기
예: 점심시간 회의 vs 개인 점심 시간 보호 갈등
타협 X: 30분씩 나누기
Win-Win O: _______
```

**방식**: 이번 주 배운 것/성장한 것 1가지

```
끝: "결국 _______를 배웠고, 이제 _______합니다"
```
