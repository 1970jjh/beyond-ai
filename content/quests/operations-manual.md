## 1. 프로그램 개요

### 1.1 핵심 콘셉트

**Beyond AI**는 사람과 AI가 동일한 직무 과제를 동시에 수행하고, 실시간으로 성과를 비교하는 **12개월 기업교육 프로그램**입니다.

```
목적: AI 시대에 인간 고유의 역량을 발견하고, AI와 공존하는 새로운 직무 역량을 개발
방식: 월 1회 AI 대전 퀘스트 + 수시 미니 챌린지
기간: 12개월 (1월-12월)
대상: 기업 임직원 전 직급 (15-30명/기수 권장)
```

### 1.2 프로그램 설계 철학

Beyond AI는 단순히 "AI보다 잘하기"를 목표로 하지 않습니다. 진짜 목표는:

1. **자기 발견**: "AI가 못하는 것이 바로 내가 잘하는 것"을 경험으로 깨닫기
2. **AI 이해**: 직접 대결해보며 AI의 실제 강점과 한계를 체험
3. **역량 성장**: 12개월간 반복 도전을 통한 직무 역량 실질 향상
4. **AI 협업 마인드**: 경쟁을 넘어 AI를 도구로 활용하는 전략적 사고

## 2. 기수 운영 구조

### 2.1 1기수 구성

| 구분 | 내용 |
|------|------|
| 기간 | 12개월 (월 1회 메인 세션) |
| 참가 인원 | 15-30명 (팀 구성 시 3-5명/팀) |
| 세션 방식 | 온라인/오프라인 혼합 |
| 퍼실리테이터 | 1명 (30명 이하), 2명 (31명 이상) |

### 2.2 월별 운영 흐름

```
[메인 퀘스트 세션] — 월 1회, 2-3시간
  - 브리핑 (5분)
  - AI 대전 수행 (30-90분, 퀘스트별 상이)
  - 결과 공개 & 디브리핑 (15-20분)
  - 성장 리플렉션 작성 (5-10분)

[미니 챌린지] — 주 1-2회, 15-20분
  - 당월 퀘스트 연관 짧은 연습
  - 개인 성장 일지 작성
  - AI와의 1:1 연습 (자율)
```

### 2.3 팀 구성 전략

**팀 구성 시 고려사항**:
- 직급 혼합 권장 (상하급자가 같은 팀에서 배움)
- 부서 간 혼합 권장 (사일로 해소 효과)
- 동질적 팀 구성이 필요한 경우: Q4 위기 대응 등 실무 밀착 과제

**팀 고정 vs 유동**:
- **권장**: 3개월 고정 → 순환 (관계 형성 + 다양성 경험 균형)

## 4. 참가자 여정 설계

### 4.1 12개월 역량 성장 커브

```
1-3월: 도구 적응기
  - AI 대전의 개념 이해
  - 자신의 현재 역량 파악
  - "AI가 나보다 잘한다"는 경험도 중요한 배움

4-6월: 전략 개발기
  - AI의 패턴/약점 파악 시작
  - 인간 강점 의도적 활용 전략 개발
  - 점수 상승 체감 시작

7-9월: 실전 역량기
  - AI와의 차별화 포인트 명확화
  - 데이터 기반 성장 측정
  - 팀 간 시너지 발현

10-12월: 통합 완성기
  - AI 협업 관점으로 전환
  - 12개월 여정의 의미 통합
  - 새로운 역할 정체성 확립
```

### 4.2 참가자 성장 측정 지표

| 지표 | 측정 방법 | 측정 시점 |
|------|---------|---------|
| 퀘스트 점수 | AI 자동 채점 | 매월 |
| AI 대비 성장률 | (사람 점수/AI 점수) 월별 추이 | 분기별 |
| 자기 효능감 | 5점 척도 설문 | 분기별 |
| AI 활용 역량 | 도구 활용 능력 테스트 | 반기별 |
| 동료 평가 점수 | 동료 피어리뷰 | 분기별 |

## 5. 플랫폼 운영 가이드

### 5.1 퀘스트 세션 생성

```
플랫폼 관리자 패널 → 새 세션 생성
  - 퀘스트 번호 선택
  - 시나리오 선택 (A/B/C/D)
  - 난이도 설정 (초급/중급/고급)
  - 참가자 초대 링크 생성
  - AI 응답 사전 생성 (선택)
```

### 5.2 AI 배틀 로직 설정

| 항목 | 초급 | 중급 | 고급 |
|------|------|------|------|
| AI temperature | 0.5 (안정적) | 0.7 (균형) | 0.3 (정확) |
| AI 응답 품질 | 70점 수준 | 78점 수준 | 88점 수준 |
| AI 약점 노출도 | 높음 | 중간 | 낮음 |

> **첫 기수 권장**: 중급으로 시작. 4개월 후 참가자 평균 점수 기반 재설정

### 5.3 데이터 관리

- 세션 종료 후 자동으로 개인/팀 데이터 저장
- 월말 자동 리포트 생성 (참가자 개인 + 팀 전체)
- 연말 종합 성장 리포트 자동 생성 (Q12 자료로 활용)

## 7. 자주 묻는 질문 (FAQ)

**Q. AI가 매번 이기면 참가자들의 동기가 떨어지지 않나요?**
A. 설계 의도입니다. "AI가 이겼을 때 무엇을 배웠는가"가 핵심입니다. 다만, AI 난이도를 참가자 수준에 맞게 조정하여 초기에는 사람이 이기는 경험도 필요합니다.

**Q. 팀별로 하나요, 개인별로 하나요?**
A. 두 방식 모두 가능합니다. 팀 방식은 협업 역량 발달에 유리하고, 개인 방식은 개인 역량 측정에 정확합니다. 격월 교차 운영을 권장합니다.

**Q. AI 결과는 어떻게 생성하나요?**
A. 플랫폼이 Claude/Gemini API를 통해 자동 생성합니다. 퍼실리테이터는 사전 검토 후 적절하지 않은 내용이 있으면 수정 가능합니다.

**Q. 참가자들이 AI 도구(ChatGPT 등)를 사용하면 어떻게 되나요?**
A. 기업별로 정책 합의가 필요합니다. 권장 정책: "AI 도구 사용 가능, 단 사용 여부 공시 + AI 활용 비율 명시". 이것 자체가 AI 협업 역량 측정의 일부가 됩니다.

**Q. 프로그램 중간에 참가자가 이탈하면?**
A. 참가자가 3개월 이상 미참여시 해당 기수 수료 취소. 다음 기수로 재시작 옵션 제공.
