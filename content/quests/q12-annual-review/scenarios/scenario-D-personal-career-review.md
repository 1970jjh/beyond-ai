# Q12 시나리오 D — 개인 커리어 1년 회고

> **배포 대상**: 참가자 (세션 시작 시 배포)
> **사용 퀘스트**: Q12 연간 회고
> **난이도 조정**: 개인 차원의 회고로 참가자들이 가장 몰입하기 쉬운 시나리오. 단, 가상 인물(강하나)의 데이터를 분석함으로써 자신의 회고에 적용할 프레임워크를 익히도록 설계.

---

## 📋 배경 브리핑

**주인공**: 강하나 (가상 인물, 29세, 데이터 마케터)
**회사**: 그로스파트너스(주) (디지털 마케팅 에이전시, 직원 85명)
**직급**: 마케팅팀 대리 (입사 3년차)
**상황**: 2024년 연말, 이직을 고민하며 자신의 1년을 진지하게 돌아보는 중

강하나는 올해 초 야심 차게 목표를 세웠습니다. 데이터 분석 역량을 키워 '마케팅 전략 수립이 가능한 시니어'로 성장하겠다는 목표였습니다. 1년이 지난 지금, 그 목표를 얼마나 달성했는지 돌아봐야 합니다.

회사에서 진행하는 공식 360도 피드백이 도착했고, 자신이 연초에 세웠던 목표와 비교할 시간이 생겼습니다. 마냥 좋지도, 마냥 나쁘지도 않은 1년이었습니다. 이제 이 데이터를 바탕으로 의미 있는 회고를 해야 합니다.

### 360도 피드백 요약 (동료 4명 + 팀장 1명 + 본인)

**동료 피드백 발췌**:

- 동료 A (3년차): *"하나 씨는 데이터 볼 때 꼼꼼하고 정확해요. 그런데 회의에서 자기 의견을 잘 안 내는 것 같아 아쉬워요. 인사이트가 있는 걸 아는데."*
- 동료 B (5년차): *"SQL이 빨리 늘었고 후배들 도와주는 게 인상적이었어요. 교육하는 거 좋아하는 것 같던데, 그쪽도 강점인 것 같아요."*
- 동료 C (2년차): *"믿고 맡길 수 있는 사람이에요. 큰 프로젝트에서 같이 일하고 싶어요."*
- 동료 D (4년차): *"가끔 너무 완벽하게 하려다가 마감 직전에 빡빡해지는 것 같아요. 80% 완성도로 빠르게 보여주는 것도 필요한 것 같아요."*

**팀장 이민재 피드백**:
*"올해 강하나 대리의 성장은 인정합니다. 특히 코스맥스 프로젝트는 탁월했어요. 그런데 커뮤니케이션에서 너무 조심스러운 게 보여요. 3년차면 좀 더 의견을 낼 때가 됐어요. 발표나 대외 커뮤니케이션을 더 적극적으로 해봤으면 좋겠어요. 내년 상반기에 팀 리드 프로젝트 기회를 줄 생각입니다."*
