## 1. 4대 평가 축

### 1.1 품질 (Quality) — 30%

| 점수 | 기준 |
|------|------|
| 5 | 주제를 완벽히 이해하고, 깊이 있는 분석과 실질적 인사이트 제공. 전문 용어를 정확히 활용하며 논거가 명확함 |
| 4 | 주제를 잘 이해하고 적절한 분석 제공. 대부분의 핵심 포인트를 커버하며 논리적 일관성 유지 |
| 3 | 주제를 이해하나 분석이 다소 표면적. 핵심 포인트 일부 누락 또는 불명확한 부분 존재 |
| 2 | 주제 이해가 부분적이고 분석이 피상적. 주요 포인트 상당 부분 누락 |
| 1 | 주제를 충분히 이해하지 못하거나 관련 없는 내용 포함. 품질이 매우 낮음 |

### 1.2 창의성 (Creativity) — 25%

| 점수 | 기준 |
|------|------|
| 5 | 완전히 독창적인 관점 또는 접근법. 예상치 못한 연결고리를 만들거나 혁신적 해결책 제시 |
| 4 | 독창적 요소가 뚜렷하며, 일반적 접근을 벗어난 아이디어 포함 |
| 3 | 일부 창의적 요소 있으나 전반적으로 표준적 접근. 참신한 시도가 부분적으로 보임 |
| 2 | 창의성이 제한적이며 대부분 기존 템플릿/프레임워크를 그대로 적용 |
| 1 | 창의성 없이 일반적이고 예측 가능한 답변만 제공 |

### 1.3 실행 가능성 (Executability) — 25%

| 점수 | 기준 |
|------|------|
| 5 | 제안한 모든 내용이 현실에서 실행 가능하며, 구체적인 실행 단계, 담당자, 일정, 예산(개략)까지 제시 |
| 4 | 대부분의 제안이 실행 가능하며, 주요 실행 방안과 고려사항이 명확함 |
| 3 | 제안의 실행 가능성은 있으나 구체성 부족. 몇 가지 현실적 장벽 미고려 |
| 2 | 제안이 이론적으로 타당하나 실행에는 상당한 장벽 존재. 구체성 심각하게 부족 |
| 1 | 비현실적이거나 실행이 불가능한 제안. 현실 여건을 전혀 고려하지 않음 |

### 1.4 시간 효율 (Time Efficiency) — 20%

| 점수 | 기준 |
|------|------|
| 5 | 제한 시간 내에 완성도 높은 결과물 제출. 시간 압박 속에서도 품질 타협 없음 |
| 4 | 제한 시간 내 제출하되 핵심 내용 모두 포함. 일부 세부 사항만 간략히 처리 |
| 3 | 제한 시간에 맞춰 제출했으나 완성도가 다소 부족하거나 일부 미완성 부분 있음 |
| 2 | 시간이 초과되었거나 제한 시간 내에 핵심 내용의 상당 부분 누락 |
| 1 | 제한 시간 대비 결과물의 완성도가 현저히 낮음 |

## 2. 최종 점수 계산

```
최종 점수 = (품질 × 0.30) + (창의성 × 0.25) + (실행가능성 × 0.25) + (시간효율 × 0.20)
         = 5점 만점
         × 20 = 100점 만점 환산
```

### 등급 기준

| 점수 | 등급 | 의미 |
|------|------|------|
| 90-100 | S (초월) | AI를 완전히 넘어섬. 인간적 가치가 극대화됨 |
| 80-89 | A (우수) | AI와 대등하거나 명확히 앞섬 |
| 70-79 | B (양호) | 일부 영역에서 AI를 능가 |
| 60-69 | C (보통) | 평균적 수준. AI와 비슷하거나 소폭 미달 |
| 50-59 | D (미흡) | 대부분 영역에서 AI에 미달 |
| 49 이하 | F (재도전) | 전반적 미흡. 재학습 후 재도전 권장 |

## 3. 퀘스트별 추가 평가 기준

일부 퀘스트는 공통 4축에 더해 **퀘스트 특화 평가 기준**이 추가됩니다.

| 퀘스트 | 추가 기준 | 가중치 조정 |
|--------|---------|------------|
| Q1 시장분석 | 데이터 인용 적절성 | 품질 +5%, 창의성 -5% |
| Q2 페르소나 | 공감도 (Empathy Score) | 창의성 +5%, 실행가능성 -5% |
| Q4 위기대응 | 의사결정 속도 | 시간효율 +10%, 창의성 -10% |
| Q6 프레젠테이션 | 청중 참여 유도 | 별도 20점 추가 (총 120점) |
| Q8 고객응대 | 감성 지능 (EQ 지표) | 창의성 +10%, 시간효율 -10% |
| Q11 갈등해결 | 중립성/공정성 | 별도 평가 항목 추가 |

## 4. AI vs 사람 비교 분석 프레임

### AI가 유리한 영역
- 데이터 처리 속도 및 정확성
- 구조화된 프레임워크 적용
- 일관성 (피로도, 기분 영향 없음)
- 방대한 지식 접근

### 사람이 유리한 영역 (채점 시 특별 주목)
- **현장 경험 기반 직관**: "이 데이터가 현실과 다른 이유는..."
- **감성적 뉘앙스**: 고객의 말 뒤에 숨은 감정 읽기
- **맥락적 판단**: 조직 문화, 인간관계, 암묵적 규칙 반영
- **창의적 도약**: 논리를 뛰어넘는 직관적 연결
- **유연한 즉흥 대응**: 예상 못한 상황에서의 임기응변

> **채점 팁**: 위 "사람이 유리한 영역"에서 실제로 AI를 넘어선 부분이 발견되면 보너스 점수(+5~10점) 부여 가능

## 5. 동료 평가 가이드라인

동료 평가는 **AI 자동 채점(70%) + 동료 평가(30%)**로 최종 점수가 결정됩니다.

### 동료 평가 시 주의사항
1. **익명성 보장**: 누구의 결과물인지 모른 채 평가
2. **근거 제시**: 점수와 함께 1-2문장 근거 반드시 작성
3. **편향 방지**: 친한 팀원에게 높은 점수, 경쟁 팀에 낮은 점수 부여 금지
4. **극단치 주의**: 모두에게 5점 또는 1점을 주는 패턴은 이상값으로 처리됨

### 동료 평가 양식

```
평가 대상: [익명 #XX]
퀘스트: [이름]

품질: [1-5] / 이유: _______
창의성: [1-5] / 이유: _______
실행가능성: [1-5] / 이유: _______

이 결과물에서 가장 인상적인 점:
___________________________

개선하면 더 좋았을 부분:
___________________________
```
