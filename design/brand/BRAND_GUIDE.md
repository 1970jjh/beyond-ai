# Beyond AI Brand Guide

## Brand Identity

### Brand Name
**Beyond AI** (비욘드 AI)

### Tagline
**"사람 vs AI, 12개월의 성장 대결"**
Sub-tagline: "AI를 넘어서는 진짜 실력을 키우다"

### Brand Story
Beyond AI는 기업 교육에 혁신적인 게이미피케이션을 도입한 프로그램이다.
학습자(사람)와 AI가 동일한 퀘스트를 수행하고, 12개월간의 성장 대결을 통해
진짜 실력을 키우는 국내 유일의 기업교육 플랫폼이다.

### Brand Personality
- **대담함 (Bold)**: 브루탈리즘 디자인으로 강렬한 첫인상
- **도전적 (Challenging)**: 사람 vs AI 대결 콘셉트
- **성장 지향 (Growth-oriented)**: 12개월 장기 퀘스트
- **투명함 (Transparent)**: 실시간 점수, 랭킹 공개

## Logo System

### Primary Logo
- **Mark**: 대괄호 안에 "B" 문자 → `[B]`
- **Wordmark**: "BEYOND AI" (Space Grotesk Bold)
- **Concept**: 대괄호는 코드/기술을, B는 Beyond를 상징
- **최소 크기**: 24px height

### Logo Variations
1. **Full Logo**: `[B] BEYOND AI` - 수평 배치
2. **Stacked Logo**: `[B]` 위에 `BEYOND AI` - 수직 배치
3. **Mark Only**: `[B]` - 파비콘, 앱 아이콘
4. **Text Only**: `BEYOND AI` - 공간 제약 시

### Logo Clear Space
- 최소 여백: Mark 높이의 50% (상하좌우)

### Logo Usage Rules
- 최소 대비: 4.5:1 (WCAG AA)
- 로고 변형/회전/왜곡 금지
- 배경과 충분한 대비 확보

## Typography

### Font Stack

#### Primary: Space Grotesk
- **용도**: 제목, 로고, 강조 텍스트
- **특징**: 기하학적 산세리프, 테크 느낌
- **Fallback**: `'Space Grotesk', 'Arial', sans-serif`

#### Secondary: Inter
- **용도**: 본문, UI 텍스트, 설명
- **특징**: 가독성 최적화된 산세리프
- **Fallback**: `'Inter', 'Helvetica Neue', sans-serif`

#### Mono: JetBrains Mono
- **용도**: 코드, 숫자, 점수, 타이머
- **특징**: 코드 에디터 느낌, 고정폭
- **Fallback**: `'JetBrains Mono', 'Consolas', monospace`

### Typography Rules
- **제목**: Space Grotesk Bold, UPPERCASE 허용
- **본문**: Inter Regular, sentence case
- **숫자/점수**: JetBrains Mono, tabular-nums
- **한글**: Pretendard로 보완 (Inter와 혼용 시)
- **줄 간격**: 본문 1.6, 제목 1.2
- **최대 줄 길이**: 65~75자 (한글 기준 32~38자)

## Motion & Animation

### Principles
- **즉각적**: 트랜지션 150~300ms
- **의미 있는**: 상태 변화를 전달하는 애니메이션만
- **브루탈한**: 부드러운 이징보다 sharp한 전환 선호

### Timing
| Type | Duration | Easing |
|------|----------|--------|
| **Micro** | 100ms | `ease-out` |
| **Standard** | 200ms | `cubic-bezier(0.25, 0, 0, 1)` |
| **Complex** | 300ms | `cubic-bezier(0.25, 0, 0, 1)` |
| **Score Pop** | 400ms | `cubic-bezier(0.68, -0.55, 0.27, 1.55)` |

### Key Animations
1. **점수 증가**: 숫자가 빠르게 올라가는 카운트업
2. **배지 획득**: 확대 → 바운스 → 정착
3. **대결 결과**: 좌(사람)/우(AI) 슬라이드 → 승패 표시
4. **퀘스트 완료**: 체크마크 드로잉 + 파티클 효과

## Voice & Tone

### 브랜드 보이스
- **직접적**: 돌려말하지 않는다
- **도전적**: "할 수 있겠어?" 느낌
- **격려하는**: 실패에도 성장을 강조
- **데이터 중심**: 감정보다 수치로 말한다

### 예시
- 로그인: "돌아왔군요. AI가 기다리고 있습니다."
- 퀘스트 시작: "이번 도전, 준비됐나요?"
- 승리: "이번엔 당신이 이겼습니다. +150점"
- 패배: "AI에게 졌지만, 실력은 올랐습니다."
- 배지 획득: "새로운 배지 획득! [배지명]"
